Sistema de Irrigação Automática (IoT)

Desenvolvi um sistema de irrigação automática utilizando um ESP32. O sistema monitora a umidade do solo e realiza a irrigação das plantas automaticamente quando necessário. Além disso, os dados coletados são enviados para um servidor e podem ser acompanhados por meio de um aplicativo.

Ver projeto no GitHub

Braço Robótico

Desenvolvi um braço robótico controlado por um controle de PS4 e por um software de gerenciamento. O sistema permite gravar os movimentos realizados pelo braço robótico para que eles possam ser reproduzidos posteriormente de forma totalmente autônoma.

Ver projeto no GitHub

Aplicativo de Leitura de Texto

Desenvolvi um aplicativo que converte textos em áudio, com o objetivo de promover maior acessibilidade para pessoas com deficiência visual e dificuldades de leitura, como a dislexia. O aplicativo aceita tanto textos digitados quanto imagens contendo texto. Utilizando tecnologias de OCR, ele extrai o conteúdo das imagens e o transforma em áudio.

Ver projeto no GitHub
